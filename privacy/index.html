Privacy Policy

Last Updated: March 20, 2026

Overview

Live Sidebar is a Chrome browser extension that displays your Twitch followed channels and YouTube subscribed channels in a unified sidebar, showing their live streaming status. This Privacy Policy describes how Live Sidebar ("we", "our", or "the Extension") collects, uses, stores, and shares your information.

Information We Collect

Authentication Data

When you connect your Twitch or YouTube account, the Extension receives OAuth tokens (access tokens and refresh tokens) that allow it to access your account data on your behalf. We do not collect or have access to your Twitch or YouTube passwords.

Data Accessed via Twitch API

  • Your Twitch user ID and display name
  • Your list of followed channels
  • Live streaming status of your followed channels (stream title, game name, viewer count, thumbnails)

Data Accessed via YouTube API

  • Your YouTube channel ID and channel title
  • Your list of subscribed channels
  • Live streaming status of your subscribed channels (stream title, viewer count, thumbnails)

Data We Do NOT Collect

  • We do not collect your browsing history or web activity
  • We do not collect personal information such as your name, email address, or physical address
  • We do not collect payment information (payments, if any, are processed entirely by third-party payment providers)
  • We do not collect analytics or usage tracking data

How We Use Your Data

All data accessed by Live Sidebar is used solely to provide the core functionality of the Extension:

  • Displaying your followed/subscribed channels and their live status in the sidebar
  • Maintaining your authenticated session with Twitch and YouTube
  • Sending desktop notifications for live stream events (if enabled by you)

We do not use your data for advertising, marketing, analytics, or any purpose unrelated to the Extension's functionality.

Data Storage

  • All data (OAuth tokens, channel lists, and cached streaming data) is stored locally on your device using Chrome's chrome.storage.local API
  • No data is transmitted to or stored on any external server operated by us
  • Twitch token exchange and refresh operations are processed through a secure proxy server that handles authentication only and does not store any user data
  • Cached channel data is automatically refreshed at regular intervals and replaced with current data

Data Sharing

We do not sell, trade, license, or share your data with any third parties.

Your data is only transmitted to:

  • Twitch API (api.twitch.tv) — to retrieve your followed channels and their live status
  • YouTube Data API (googleapis.com) — to retrieve your subscribed channels and their live status
  • YouTube RSS Feeds (youtube.com/feeds) — to check for recent videos (no authentication required, no user data transmitted)

Data Retention and Deletion

  • All data is stored locally on your device and is not retained by us on any server
  • You can delete all stored data at any time by:
    • Clicking the "Disconnect" button for each platform within the Extension
    • Uninstalling the Extension (which removes all locally stored data)
  • Upon disconnecting an account, all associated tokens and cached data for that platform are immediately deleted from local storage

Google API Services — Limited Use Disclosure

Live Sidebar's use and transfer to any other app of information received from Google APIs will adhere to the Google API Services User Data Policy, including the Limited Use requirements.

  • We only use YouTube API data to display your subscribed channels and their live streaming status
  • We do not transfer YouTube API data to any third party
  • We do not use YouTube API data for advertising or any unrelated purpose
  • We do not allow humans to read YouTube API data except as necessary to provide the Extension's functionality to you

YouTube API Services

Live Sidebar uses YouTube API Services. By using Live Sidebar, you also agree to be bound by the YouTube Terms of Service.

You can revoke Live Sidebar's access to your YouTube data at any time through your Google Account permissions page.

Twitch API Services

Live Sidebar uses Twitch API Services. By using Live Sidebar, you also agree to be bound by the Twitch Terms of Service and the Twitch Developer Services Agreement.

You can revoke Live Sidebar's access to your Twitch data at any time through your Twitch Connections settings.

Children's Privacy

Live Sidebar is not directed at children under the age of 13 (or applicable age in your jurisdiction). We do not knowingly collect personal information from children.

Changes to This Privacy Policy

We may update this Privacy Policy from time to time. Any changes will be reflected by updating the "Last Updated" date at the top of this page. Your continued use of the Extension after changes are posted constitutes your acceptance of the updated Privacy Policy.

Contact

If you have questions about this Privacy Policy or your data, please contact us.

プライバシーポリシー

最終更新日: 2026年3月20日

概要

Live Sidebarは、TwitchのフォローチャンネルとYouTubeの登録チャンネルのライブ配信状況を統合サイドバーに表示するChrome拡張機能です。本プライバシーポリシーは、Live Sidebar（以下「当拡張機能」）がお客様の情報をどのように収集、使用、保存、共有するかについて説明します。

収集する情報

認証データ

TwitchまたはYouTubeアカウントを連携する際、当拡張機能はOAuthトークン（アクセストークンおよびリフレッシュトークン）を受け取ります。お客様のTwitchまたはYouTubeのパスワードを収集したり、アクセスすることはありません。

Twitch APIを通じてアクセスするデータ

  • TwitchユーザーIDおよび表示名
  • フォロー中のチャンネル一覧
  • フォロー中チャンネルのライブ配信状況（配信タイトル、ゲーム名、視聴者数、サムネイル）

YouTube APIを通じてアクセスするデータ

  • YouTubeチャンネルIDおよびチャンネル名
  • 登録チャンネル一覧
  • 登録チャンネルのライブ配信状況（配信タイトル、視聴者数、サムネイル）

収集しないデータ

  • 閲覧履歴やウェブ上の行動は収集しません
  • 氏名、メールアドレス、住所などの個人情報は収集しません
  • 決済情報は収集しません（決済がある場合、すべて第三者の決済プロバイダーが処理します）
  • 分析データや利用追跡データは収集しません

データの使用方法

Live Sidebarがアクセスするすべてのデータは、当拡張機能のコア機能を提供するためにのみ使用されます：

  • フォロー/登録チャンネルとそのライブ状況をサイドバーに表示
  • TwitchおよびYouTubeとの認証セッションの維持
  • ライブ配信開始時のデスクトップ通知の送信（有効にした場合）

広告、マーケティング、分析、その他当拡張機能の機能に関係のない目的でデータを使用することはありません。

データの保存

  • すべてのデータ（OAuthトークン、チャンネル一覧、キャッシュされた配信データ）はChromeのchrome.storage.local APIを使用してお客様のデバイスにローカル保存されます
  • 当方が運営する外部サーバーにデータが送信・保存されることはありません
  • Twitchのトークン交換とリフレッシュは、認証のみを処理しユーザーデータを保存しないセキュアなプロキシサーバーを介して行われます
  • キャッシュされたチャンネルデータは定期的に自動更新され、最新のデータに置き換えられます

データの共有

お客様のデータを第三者に販売、取引、ライセンス供与、共有することはありません。

データの送信先は以下に限定されます：

  • Twitch API（api.twitch.tv）— フォローチャンネルとライブ状況の取得
  • YouTube Data API（googleapis.com）— 登録チャンネルとライブ状況の取得
  • YouTube RSSフィード（youtube.com/feeds）— 最新動画の確認（認証不要、ユーザーデータの送信なし）

データの保持と削除

  • すべてのデータはお客様のデバイスにローカル保存され、当方のサーバーには保持されません
  • 以下の方法でいつでもすべての保存データを削除できます：
    • 拡張機能内の各プラットフォームの「連携解除」ボタンをクリック
    • 拡張機能のアンインストール（ローカル保存データがすべて削除されます）
  • アカウントの連携を解除すると、そのプラットフォームに関連するすべてのトークンとキャッシュデータがローカルストレージから即座に削除されます

Google APIサービス — 限定使用の開示

Live SidebarによるGoogle APIから受信した情報の使用および他のアプリへの転送は、限定使用の要件を含むGoogle APIサービスのユーザーデータポリシーに準拠します。

  • YouTube APIデータは、登録チャンネルとそのライブ配信状況の表示にのみ使用します
  • YouTube APIデータを第三者に転送しません
  • YouTube APIデータを広告やその他無関係な目的に使用しません
  • 拡張機能の機能を提供するために必要な場合を除き、人間がYouTube APIデータを閲覧することを許可しません

YouTube APIサービス

Live SidebarはYouTube APIサービスを使用しています。Live Sidebarを使用することにより、YouTube利用規約にも同意したものとみなされます。

Googleアカウントの権限ページから、いつでもLive SidebarのYouTubeデータへのアクセスを取り消すことができます。

Twitch APIサービス

Live SidebarはTwitch APIサービスを使用しています。Live Sidebarを使用することにより、Twitch利用規約およびTwitch開発者サービス契約にも同意したものとみなされます。

Twitchの接続設定から、いつでもLive SidebarのTwitchデータへのアクセスを取り消すことができます。

児童のプライバシー

Live Sidebarは13歳未満（またはお住まいの地域の該当年齢未満）の児童を対象としていません。児童の個人情報を故意に収集することはありません。

プライバシーポリシーの変更

本プライバシーポリシーは随時更新される場合があります。変更はページ上部の「最終更新日」の更新により反映されます。変更の掲載後も当拡張機能を継続して使用することにより、更新されたプライバシーポリシーに同意したものとみなされます。

お問い合わせ

本プライバシーポリシーまたはお客様のデータに関するご質問は、お問い合わせフォームよりご連絡ください。

隐私政策

最后更新：2026年3月20日

概述

Live Sidebar 是一款 Chrome 浏览器扩展程序，可在统一的侧边栏中显示您关注的 Twitch 频道和 YouTube 订阅频道的直播状态。本隐私政策描述了 Live Sidebar（以下简称"我们"或"本扩展程序"）如何收集、使用、存储和共享您的信息。

我们收集的信息

认证数据

当您连接 Twitch 或 YouTube 帐户时，本扩展程序会接收 OAuth 令牌（访问令牌和刷新令牌），以代表您访问帐户数据。我们不会收集或访问您的 Twitch 或 YouTube 密码。

通过 Twitch API 访问的数据

  • 您的 Twitch 用户 ID 和显示名称
  • 您关注的频道列表
  • 关注频道的直播状态（直播标题、游戏名称、观看人数、缩略图）

通过 YouTube API 访问的数据

  • 您的 YouTube 频道 ID 和频道名称
  • 您的订阅频道列表
  • 订阅频道的直播状态（直播标题、观看人数、缩略图）

我们不收集的数据

  • 我们不收集您的浏览历史或网络活动
  • 我们不收集姓名、电子邮件地址或实际地址等个人信息
  • 我们不收集支付信息（如有支付，均由第三方支付提供商处理）
  • 我们不收集分析或使用追踪数据

数据使用方式

Live Sidebar 访问的所有数据仅用于提供本扩展程序的核心功能：

  • 在侧边栏中显示您关注/订阅的频道及其直播状态
  • 维护您与 Twitch 和 YouTube 的认证会话
  • 发送直播事件的桌面通知（如您已启用）

我们不会将您的数据用于广告、营销、分析或与本扩展程序功能无关的任何目的。

数据存储

  • 缓存的频道数据会定期自动刷新并替换为最新数据

数据共享

我们不会向任何第三方出售、交易、许可或共享您的数据。

您的数据仅传输至：

  • Twitch API（api.twitch.tv）— 获取您关注的频道及其直播状态
  • YouTube Data API（googleapis.com）— 获取您订阅的频道及其直播状态
  • YouTube RSS 源（youtube.com/feeds）— 检查最新视频（无需认证，不传输用户数据）

数据保留和删除

  • 所有数据存储在您的设备本地，不会保留在我们的任何服务器上
  • 您可以随时通过以下方式删除所有存储的数据：
    • 点击扩展程序内各平台的"断开连接"按钮
    • 卸载扩展程序（将删除所有本地存储的数据）
  • 断开帐户连接后，该平台的所有相关令牌和缓存数据将立即从本地存储中删除

개인정보 처리방침

최종 업데이트: 2026년 3월 20일

개요

Live Sidebar는 팔로우 중인 Twitch 채널과 구독 중인 YouTube 채널의 라이브 방송 상태를 통합 사이드바에 표시하는 Chrome 브라우저 확장 프로그램입니다. 본 개인정보 처리방침은 Live Sidebar(이하 "당사" 또는 "본 확장 프로그램")가 사용자의 정보를 수집, 사용, 저장 및 공유하는 방법을 설명합니다.

수집하는 정보

인증 데이터

Twitch 또는 YouTube 계정을 연결하면 본 확장 프로그램은 사용자를 대신하여 계정 데이터에 액세스할 수 있는 OAuth 토큰(액세스 토큰 및 리프레시 토큰)을 수신합니다. 당사는 사용자의 Twitch 또는 YouTube 비밀번호를 수집하거나 액세스하지 않습니다.

Twitch API를 통해 액세스하는 데이터

  • Twitch 사용자 ID 및 표시 이름
  • 팔로우 중인 채널 목록
  • 팔로우 중인 채널의 라이브 방송 상태 (방송 제목, 게임 이름, 시청자 수, 썸네일)

YouTube API를 통해 액세스하는 데이터

  • YouTube 채널 ID 및 채널 이름
  • 구독 채널 목록
  • 구독 채널의 라이브 방송 상태 (방송 제목, 시청자 수, 썸네일)

수집하지 않는 데이터

  • 브라우징 기록이나 웹 활동을 수집하지 않습니다
  • 이름, 이메일 주소, 실제 주소 등 개인 정보를 수집하지 않습니다
  • 결제 정보를 수집하지 않습니다 (결제가 있는 경우 전적으로 제3자 결제 제공업체가 처리합니다)
  • 분석 또는 사용 추적 데이터를 수집하지 않습니다

데이터 사용 방법

Live Sidebar가 액세스하는 모든 데이터는 본 확장 프로그램의 핵심 기능을 제공하기 위해서만 사용됩니다:

  • 사이드바에 팔로우/구독 채널과 라이브 상태 표시
  • Twitch 및 YouTube와의 인증 세션 유지
  • 라이브 방송 이벤트에 대한 데스크톱 알림 전송 (사용자가 활성화한 경우)

당사는 광고, 마케팅, 분석 또는 본 확장 프로그램의 기능과 관련 없는 목적으로 데이터를 사용하지 않습니다.

데이터 저장

  • 모든 데이터(OAuth 토큰, 채널 목록, 캐시된 방송 데이터)는 Chrome의 chrome.storage.local API를 사용하여 사용자의 기기에 로컬 저장됩니다
  • 당사가 운영하는 외부 서버로 데이터가 전송되거나 저장되지 않습니다
  • Twitch 토큰 교환 및 갱신 작업은 인증만 처리하고 사용자 데이터를 저장하지 않는 보안 프록시 서버를 통해 처리됩니다
  • 캐시된 채널 데이터는 정기적으로 자동 갱신되어 최신 데이터로 대체됩니다

데이터 공유

당사는 사용자의 데이터를 제3자에게 판매, 거래, 라이선스 부여 또는 공유하지 않습니다.

데이터는 다음으로만 전송됩니다:

  • Twitch API (api.twitch.tv) — 팔로우 채널 및 라이브 상태 조회
  • YouTube Data API (googleapis.com) — 구독 채널 및 라이브 상태 조회
  • YouTube RSS 피드 (youtube.com/feeds) — 최신 동영상 확인 (인증 불필요, 사용자 데이터 전송 없음)

데이터 보존 및 삭제

  • 모든 데이터는 사용자의 기기에 로컬 저장되며 당사의 서버에 보존되지 않습니다
  • 다음 방법으로 언제든지 모든 저장된 데이터를 삭제할 수 있습니다:
    • 확장 프로그램 내 각 플랫폼의 "연결 해제" 버튼 클릭
    • 확장 프로그램 제거 (모든 로컬 저장 데이터가 삭제됩니다)
  • 계정 연결을 해제하면 해당 플랫폼과 관련된 모든 토큰 및 캐시 데이터가 로컬 저장소에서 즉시 삭제됩니다

Google API 서비스 — 제한적 사용 공개

Live Sidebar의 Google API에서 수신한 정보의 사용 및 다른 앱으로의 전송은 제한적 사용 요구 사항을 포함한 Google API 서비스 사용자 데이터 정책을 준수합니다.

  • YouTube API 데이터는 구독 채널 및 라이브 방송 상태 표시에만 사용합니다
  • YouTube API 데이터를 제3자에게 전송하지 않습니다
  • YouTube API 데이터를 광고 또는 관련 없는 목적으로 사용하지 않습니다
  • 확장 프로그램의 기능을 제공하기 위해 필요한 경우를 제외하고는 사람이 YouTube API 데이터를 열람하는 것을 허용하지 않습니다
